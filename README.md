# Warat Chumin Portfolio

Senior Fullstack Software Engineer Portfolio focusing on Scalable Systems, Architecture, and Performance.

## 🚀 Tech Stack
- **Frontend**: Next.js 15+, React 19, Tailwind CSS
- **Design**: Vanilla CSS, Framer Motion, Lucide Icons
- **Language**: Bilingual (Thai / English)
- **Deployment**: Vercel

## 🛠 Features
- **Scalable Architecture Focus**: Highlighted expertise in High-Throughput systems.
- **Bilingual Support**: Full localized content in TH/EN via custom `LanguageProvider`.
- **Dynamic Projects Gallery**: Tailored display logic for featured vs. list view.
- **Premium Design**: Dark mode support with a custom Red accent theme.

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## 📂 Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components/ui`: Reusable UI components (ProjectCard, Section, etc).
- `src/constants`: Static content and project data.
- `src/lib`: Utility functions.
- `docs/context.md`: Detailed history of the project's evolution and design decisions.
